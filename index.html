Holiday Destination

The application can help user to search for their next holiday destination city and point of interest in it.
The user can select destination city and one of the tourist attractions in the area.
Museums, Accommodation, Bars and Restaurants.
Only one of POI can be selected at the time.
By selecting one of POI and clicking Show button the markers will be placed on the map. Each marker can be clicked to show basic information about the place.
